==Description==
Blue Haze WordPress theme, Copyright (c) 2023 Elite-Star-Services.com

Blue Haze Uses Bootstrap https://getbootstrap.com/ licensed under Apache 2.0.
Blue Haze Uses Font Awesome https://fontawesome.com/ licensed under MIT.

Blue Haze is based on Bootstrap Basic v3 by Vee Winch.
Bootstrap Basic was built using Twitter Bootstrap v3

Blue Haze is a developer friendly, highly configurable theme for the classic editor.
This theme can use all Bootstrap classes, elements and styles.
